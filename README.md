# Quark Game Engine

Quark is my personal learning project.


## Environment

- Windows, MacOS
- C++17+( have been tested on MSVC and Clang)
- CMake 3.10 +
- Vulkan1.3+

## Feature

- Render Hardware Interface
- Vulkan Backend
- 3D Scene Renderer (Forward Renderer)
- Entity Component System
- Asset System
- Event System
- UI with ImGui
- Scene Editor Tool
